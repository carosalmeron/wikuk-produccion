# wikuk · Control de Producción

## Instalación y arranque local

```bash
npm install
npm start
```

Se abre en http://localhost:3000

## Despliegue en Vercel (gratis)
1. Crea cuenta en vercel.com
2. Instala Vercel CLI: npm install -g vercel
3. Desde esta carpeta: vercel
4. Sigue las instrucciones — en ~2 minutos tienes la URL

## Despliegue en Netlify (gratis)
1. npm run build
2. Arrastra la carpeta /build a netlify.com/drop
